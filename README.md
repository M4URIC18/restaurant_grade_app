# Restaurant Grade App

A machine learning application for predicting restaurant grades.

## Project Structure

```
restaurant-grade-app/
├── app.py              # Main application file
├── src/
│   ├── __init__.py
│   ├── predictor.py    # Prediction logic
│   ├── data_loader.py  # Data loading utilities
│   └── utils.py        # Helper functions
├── models/
│   └── grade_model.pkl # Trained model (to be added)
├── data/               # Dataset files (to be added)
├── requirements.txt    # Python dependencies
└── README.md          # This file
```

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Add your trained model to `models/grade_model.pkl`

3. Add your cleaned datasets to the `data/` directory
